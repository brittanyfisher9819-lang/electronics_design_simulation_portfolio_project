# KiCad Project Notes

This folder stands in for a KiCad project and is suitable for a GitHub portfolio upload when you want to demonstrate structure and engineering thinking.

## Intended Files in a future expanded version

- `sensor_interface.kicad_pro`
- `sensor_interface.kicad_sch`
- `sensor_interface.kicad_pcb`
- fabrication outputs
- BOM export

## Layout Intent

- Keep analog front-end traces short
- Place decoupling capacitors near regulator and op-amp supply pins
- Separate noisy power input section from signal conditioning section
- Use a ground pour for return path stability
